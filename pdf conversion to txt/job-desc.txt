Senior Backend Engineer - AI/ML Focus
InnovateTech Solutions | San Francisco, CA | Full-time

COMPANY OVERVIEW
InnovateTech Solutions is a leading technology company specializing in AI-powered enterprise software solutions. We serve Fortune 500 companies with cutting-edge applications that leverage machine learning and natural language processing to solve complex business problems.

JOB DESCRIPTION
We are seeking a Senior Backend Engineer with strong expertise in Python, Django, and machine learning to join our growing engineering team. The ideal candidate will have experience building scalable backend systems and implementing AI/ML solutions for production environments.

KEY RESPONSIBILITIES
• Design and develop robust backend services using Python and Django framework
• Build and maintain RESTful APIs for data processing, retrieval, and manipulation
• Work with large-scale PostgreSQL databases, optimizing queries and ensuring high performance
• Implement and maintain vector databases (ChromaDB, Weaviate, or similar) for similarity search applications
• Fine-tune pre-trained embedding models including Sentence Transformers, BERT, and other transformer architectures
• Develop semantic search capabilities and document matching systems for enterprise clients
• Collaborate with ML engineers to deploy and scale machine learning models in production
• Optimize API response times and system performance for high-traffic applications
• Participate in code reviews, architectural decisions, and technical documentation
• Mentor junior developers and contribute to team knowledge sharing

REQUIRED QUALIFICATIONS
• 5+ years of experience in backend development with Python and Django
• Strong experience with PostgreSQL database design, optimization, and query performance
• Hands-on experience with machine learning frameworks (TensorFlow, PyTorch, scikit-learn)
• Experience fine-tuning pre-trained NLP models, particularly transformer models like BERT, Sentence Transformers
• Knowledge of vector databases and similarity search technologies (ChromaDB, Weaviate, Pinecone, or similar)
• Proficiency in building and consuming RESTful APIs
• Experience with cloud platforms (AWS, GCP, or Azure)
• Strong understanding of software engineering best practices and design patterns
• Excellent problem-solving skills and attention to detail
